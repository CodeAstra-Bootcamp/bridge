Curriculum

Computational Concepts

Problem Solving

You will learn solving problems using computers. You will dive into data structures and algorithms using C and Java. You will also look into Object Oriented Programming and various Design Patterns to model your solutions to various problems. We will also give you assignments .Our mentors would be helping you solve the assignments.

Core Computer Concepts

You will be given access to Ravindra Babu's complete set of GATE videos on Operating Systems, Computer Networks and Database Management System.

Introduction to Machine Learning and Big Data

You will develop intelligent applications that use machine learning algorithms to derive intelligence from data. We will use Hadoop to handle big data.
